# 📝 Prueba

Código para la prueba del tercer módulo "JavaScript para la web" del curso de
Desarrollo Full Stack JavaScript de Desafío Latam

## 🔧 Stack

- **Vite:** 🚀 Un poco overkill para la simplicidad de este proyecto,
  pero permite integrar todo más sencillamente
- **UnoCSS:** 🎨 Tailwind, pero más simple.
- **Apache Echarts:** 📈 Para gráficos
- **Luxon:** 📅 Para manejar fechas
